# Frontend Design Skill

When editing this website's design:

- Keep the existing visual identity.
- Do not create generic AI-looking UI.
- Use the existing colors, spacing, and typography when possible.
- Make small, intentional design changes.
- Improve readability, alignment, hierarchy, and mobile responsiveness.
- Keep the style modern, gaming-focused, and professional.
- Make calls to action clear without making the site look spammy.
- Avoid random gradients, excessive shadows, and unnecessary animations.
- Avoid duplicated CSS.
- Check that the page still works on mobile.
- Before saying done, explain what changed visually and which files changed.
